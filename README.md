    <article class="markdown-body entry-content" itemprop="text"><h1><a id="user-content-remote-control-car" class="anchor" aria-hidden="true" href="#remote-control-car"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Remote Control Car</h1>
<ol>
<li>
<p>Test Motor Code</p>
<ul>
<li>Copy the code from Combined_Driver_board_test.ino and paste it onto your Arduino sketch.</li>
<li>This code will check if you have wired your motors correctly. Your car will move forward and backwards.</li>
<li>If your car is running in circles, change either side of the motor wire on the motor driver (the red base).</li>
</ul>
</li>
<li>
<p>IR Receiver</p>
<ul>
<li>Download IRREMOTE.ZIP onto your laptop. The is the library required for the IR codes.</li>
<li>Include library into your code on Arduino: Sketch → Include Library → Add ZIP Library → … → IRREMOTE.ZIP</li>
<li>Copy the code from IR_Read.ino to your Arduino sketch.</li>
<li>This code allows you to decode values from your remote. You can now pick up what each button on the remote will be decoded as!</li>
<li>Open your Serial Monitor to read the HEX values.</li>
</ul>
</li>
<li>
<p>Official Code</p>
<ul>
<li>Copy codes from Car_controller.ino to your Arduino sketch.</li>
<li>This is the code that combines both the motor and IR module.</li>
<li>You can now remotely control your car :)</li>
<li>Car_controller.ino moves your car forward, backward and stop.</li>
<li>Car_controller0.3.ino moves your car forward, backward, left, right and stop.</li>
</ul>
</li>
</ol>
